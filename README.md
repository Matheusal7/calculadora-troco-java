# Calculadora de Troco em Java ☕

Este é um dos meus primeiros projetos estudando lógica de programação com Java. 
O objetivo foi praticar a entrada de dados do usuário e operações matemáticas simples.

## 🚀 Funcionalidades
- Recebe o valor de um produto.
- Recebe o valor pago pelo cliente.
- Calcula e exibe o troco exato.

## 🛠️ Tecnologias
- Java (JDK 17 ou superior)
- Scanner API para entrada de dados

## 📖 O que aprendi
Neste exercício, foquei em entender como o Java lida com tipos de dados numéricos (`double`) e como estruturar uma sequência lógica de comandos.
